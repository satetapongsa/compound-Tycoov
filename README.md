# Anmum Core (Sci-Fi Tycoon)

An advanced Sci-Fi Dashboard Idle Tycoon game built entirely with Vanilla HTML, CSS, and JavaScript.

## Features
- **Hacking Core**: Click to manually generate Credits (C$).
- **Infrastructure**: Purchase Data Nodes and Server Farms to generate passive income.
- **Galactic Exchange**: Trade tech stocks (e.g., Quantum Bank, Astro Tech) that yield dividends and randomly fluctuate in price.
- **Dynamic Events**: Random events like Tech Booms or Solar Flares that temporarily affect your income or stock prices.
- **Tech Upgrades**: Install advanced systems like AI Traders and ICE Firewalls.
- **Live Performance Matrix**: Real-time charts powered by Chart.js.

Simply open `index.html` in your browser to play.
